Neostralis | Q3

Q3 Dokumentation | API Dokumentation

Neostralis ist ein kleiner Minecraft Server welcher es aber, vor allem Entwickler technisch, in sich hat. Seit schon ungefähr zwei Jahren arbeite ich in meiner Freizeit von Zeit zu Zeit für diesen Server. Tatsächlich habe ich auch keine Erfahrung im PlugIns schreiben mit Java, sondern code ausschließlich in der hauseigenen, vom Owner selber entwickelten, Programmiersprache. Q3 ist eine extrem mächtige Programmiersprache welche es dem Entwickler erlaubt schnell code lokal zu schreiben und auf Knopfdruck live und ohne reloads auf dem Server laufen zu lassen.

q3Extension

VS Marktplatz | GitHub

Wenn man viel Programmiert weiß man etwas: Ohne Syntax Highlighting geht gar nichts. Komplexe Code abschnitte werden extrem schwer zu lesen und man arbeitet deutlich langsamer. Aus diesem Grund habe ich mich schlau gemacht und eine kleine Visual Studio COde Erweiterung erstellt. Mit der Hilfe von dieser hat man Syntaxhighliting für alle q3 Dateien in VS Code.
